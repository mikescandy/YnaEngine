WaitScript Class (Yna.Engine.Script)

Wait the given amount of miliseconds. Use this script to create time gaps in your animations. Note that the time waited may not be accurate. As updates are made at a certain framerate by the game container, the wait time may go beyond it's defined value (about a few miliseconds)
