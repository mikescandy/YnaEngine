​Yna is a lightweight 2D and 3D game engine using MonoGame Framework (XNA is only supported for Windows Phone 7 & 8) . The goal of this project is to give the developer the ability to create games in 2D or 3D easily on multiple platforms such as Windows Phone, Windows 8, or Linux. Yna is not a complex engine compared to its competitor but suitable for all developers who want an easy way to quickly create a prototype or a simple game.
It is mainly inspired by Flixel, an ActionScript 3 Framework for 2D part and a bit by Three.js for 3D.
 
Download

2D Engine

  • Tiled map 2D and basic isometric map 2D
  • QuadTree for optimised collision detection
  • Entity, Sprite (static, animated)
  • Scene (With some effects like shake)
  • GUI manager with buttons, progress bar, etc..
  • Components like Virtual pad (usefull for mobile development)

3D Engine

  • Custom gemotries (Cube, Cylinder, Plane and more)
  • Materials with various effects (include custom effect)
  • Camera FPS, TPS and fixed
  • Camera controller (with key mapping)
  • FBX and X 3d models (more formats are planned to be supported)

Platforms

  • Windows XP, Vista, 7 and 8
  • Windows 8 & WindowsRT
  • Windows Phone 7 (XNA)
  • Linux
  • Mac OSX (not officially supported)
